# Ronin-electron

[Ronin](https://github.com/hundredrabbits/Ronin) running in electron.

![Screenshot](/media/RoninScreenshot.png)

For building you need git and node install.

Only linux install script at this monent, but it use [electron-packager](https://github.com/electron-userland/electron-packager), so you can build on any platform or wait for other os install scripts.
